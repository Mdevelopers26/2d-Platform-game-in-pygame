How to run the game:

1.	Unzip the file
2. 	Ensure all of the supporting audio/image files are in the same directory as the python file
3. 	Run the CrashLanded.py file, either by command line or through an IDE

To see the code in full, please open the .py file with any IDE. Running the code will start the game, providing the supporting files
are stored in the same directory.

Please see our description for the lore and basic controls for playing the game!

Modules required to run the game:

1. Pygame
2. Math
3. Time
4. Random
